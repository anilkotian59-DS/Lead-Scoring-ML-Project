# Lead Scoring ML Project

## Objective
Predict the probability of lead conversion using machine learning.

## Models Built
- Logistic Regression
- Random Forest
- XGBoost (Final Model)

## Final Model
Tuned XGBoost with 0.65 decision threshold.

ROC-AUC: ~0.90  
Precision: ~83%

## Business Impact
Improves sales efficiency by prioritising high-quality leads.
